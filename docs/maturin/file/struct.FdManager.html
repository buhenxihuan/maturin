pub struct FdManager {
    files: Vec<Option<Arc<dyn File>>>,
    fd_allocator: FdAllocator,
    limit: usize,
    umask: i32,
}
Expand description

文件描述符管理，每个进程应该有一个 这个结构 Drop 时会自动释放文件的 Arc

Fields§

§files: Vec<Option<Arc<dyn File>>>

内部包含的文件

§fd_allocator: FdAllocator

描述符和分配器

§limit: usize

最大 fd 限制

§umask: i32

创建文件时的的 mode 需要屏蔽这些位 如 sys_open 时权限为 0o666 ，而调用者的 umask 为 0o022，则实际权限为 0o644

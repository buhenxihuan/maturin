    空文件，用于 dev/null
  • temp 🔒
    临时性文件，语义上来说，OS可以定期清理其中文件，不保证永久保存
  • virt_dir 🔒
    虚拟文件系统的目录。不需要考虑把数据塞进页里
  • virt_file 🔒
    内存中保存的虚文件，可以有 backend 指向它实际映射的内容

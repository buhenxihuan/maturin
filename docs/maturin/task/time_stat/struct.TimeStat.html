Expand description

进程的时间统计，基于 lmbench 需要，主要用于 sys_getrusage

Fields§

§tid: usize

线程 id。因为到时间时会自动发送信号，所以最好另外存一份pid在这

§utime_us: usize

用户态经过时间

§stime_us: usize

内核态经过时间

§user_tick: usize

进入用户态时标记当前系统时间，退出时累加统计

§kernel_tick: usize

进入内核态时标记当前系统时间，退出时累加统计

§start_tick: usize

开始运行时的系统时间

§timer_type: TimerType

计时器类型

§timer_interval_us: usize

设置下一次触发计时器的区间

当 timer_remained_us 归零时，**如果 timer_interval_us 非零 **，则将其重置为 timer_interval_us 的值； 否则，则这个计时器不再触发

§timer_remained_us: usize

当前计时器还剩下多少时间。

根据 timer_type 的规则不断减少，当归零时触发信号

Module maturin::syscall::futex

source ·
Expand description

专门处理 linux 特色的 futex 锁。

在 rCore-Tutorial 中，把锁、信号量、条件变量机制分别用很多不同的 syscall 来实现， 而在 linux 中都浓缩在了以 futex 为核心的几个 syscall 上，并统一了语义， 具体的机制区别由用户态的库完成，只有当发送冲突时才进入内核。

由于 futex 参数复杂，所以特别开了一个子模块来放和它相关的实现与 flag
